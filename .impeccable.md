# Triptych Design Context

## Users

Researchers solving hard problems — math, physics, data analysis, writing. Primary user works in extended sessions at a desk, but the design must support multi-device setups: multiple browser tabs across a computer and iPad simultaneously (e.g., drawing on iPad while chat + display live on desktop monitors). Desktop and iPad are the primary targets. The interface is a workspace, not a product to be marketed.

## Brand Personality

**Simple and powerful and mechanical.**

Like a well-made lab instrument — everything has a place, nothing is decorative, the materials are good. Not cold or sterile, but not warm or cozy either. A clean workbench with precision tools laid out where you need them.

## Aesthetic Direction

**Tone**: Focused calm. Organized lab bench. The interface should recede when you're working and surface controls only when you reach for them.

**Reference**: Claude.ai — likes the typography, spacing, color sensibility. But cleaner/less cluttered than their marketing pages. The product UI parts of claude.ai, not the landing page.

**Anti-references**:
- Aggressively dark mode with neon accents and glowing effects
- Heavily themed interfaces where the theme is louder than the content
- Overly light/bright interfaces that feel sterile
- Generic "AI tool" aesthetic (purple gradients, cyan-on-dark, sparkle effects)

**Theme**: Dark — researchers work extended sessions, often across day and evening. But a *comfortable* dark, not an aggressive one. Surfaces should have enough differentiation to create depth without high contrast. Tint neutrals warm.

**Accent color**: Keep the copper/warm-orange family (connects to Claude's brand), but soften it — less saturated, more of a muted terracotta than a neon copper. The current #DA7756 reads a bit hot. Something calmer.

**Typography**: Move away from all-monospace. Use a refined sans-serif for UI chrome, labels, and body text. Reserve monospace for the terminal and code only. Claude.ai's typographic approach is the reference — clean, well-spaced, good hierarchy.

## Design Principles

1. **Tools, not decoration** — every element serves a function. If it doesn't help you work, remove it. No ornamental borders, shadows, or effects.

2. **Quiet until needed** — the interface recedes during focused work. Controls surface on hover or interaction, not all at once. Progressive disclosure over visual clutter.

3. **Device-agnostic** — works on a 27" monitor and an iPad in portrait. Panels adapt gracefully. Touch targets are generous enough for stylus/finger input without looking like a mobile app on desktop.

4. **Precision spacing** — consistent rhythm using a 4pt scale. Generous whitespace between sections, tight grouping within related elements. The spacing *is* the design.

5. **Warm mechanical** — neutrals tinted slightly warm. The copper accent is a material choice (like brass hardware on a wooden instrument), not a brand splash. Use it sparingly — connection status, active states, the occasional confirmation. 90% of the interface should be neutral.
